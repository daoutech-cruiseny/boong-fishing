# 고고 다우 낚시터 🎣

입을 **'O'로 벌리면 붕어가 되어 낚시**하는 사내 웹캠 게임. 카메라 + MediaPipe 얼굴 인식으로
**붕어 모드 → 캐스팅(고개 회전) → 입질 대기 → 뻐끔 챔질**의 낚시 루프를 플레이합니다.
보상은 전부 가짜(재미용)이고, 어항·랭킹은 브라우저에 로컬 저장됩니다.

> Claude Design에서 만든 디자인(저수지 씬 · Jua/Gaegu 폰트 · 코랄 #E2584A · 등급 색)을
> 그대로 적용한 **빌드 단계 없는 정적 웹앱**입니다. 그대로 정적 호스팅에 올리면 됩니다.

## 구조

```
boong-fishing/
├─ index.html        앱 셸 (입장 / 캘리브레이션 / 로비 + 어항·랭킹·설정 모달)
├─ css/styles.css    디자인 토큰 + 전체 화면·컴포넌트 스타일
└─ js/
   ├─ face.js        MediaPipe Face Landmarker 래퍼 (카메라·지표·캘리브레이션)
   ├─ game.js        순수 로직 (아이템 카탈로그·확률·난이도·어항 저장·랭킹/NPC)
   └─ main.js        컨트롤러 (화면 전환·게임 상태머신·모달·설정·폴백)
```

## 로컬 실행

ES 모듈을 쓰기 때문에 `file://`로 직접 열면 안 되고, **로컬 서버**가 필요합니다.
카메라(`getUserMedia`)는 `localhost`에서는 보안 컨텍스트로 인정되어 잘 동작합니다.

```bash
cd boong-fishing
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속 → 카메라 권한 허용
```

(또는 `npx serve`, VS Code Live Server 등 어떤 정적 서버든 가능)

## 배포

- 정적 파일만 있으므로 **GitHub Pages / Netlify / Vercel / Cloudflare Pages** 등 어디든 그대로 올리면 됩니다.
- **운영에서는 HTTPS 필수**입니다. 카메라 권한은 보안 컨텍스트(HTTPS 또는 localhost)에서만 허용됩니다.
- MediaPipe(런타임 라이브러리·WASM·모델)는 **CDN에서 실행 시 로드**합니다(jsdelivr + storage.googleapis.com).
  첫 실행 때 모델(수 MB)을 내려받으므로 인터넷 연결이 필요합니다.
  - 사내망에서 외부 CDN이 막혀 있으면, `js/face.js`의 `WASM_URL`/`ESM_URL`/`MODEL_URL`을
    자체 호스팅 경로로 바꾸고 해당 파일을 함께 배포하세요.

## 조작 / 폴백

| 단계 | 얼굴(카메라) | 폴백(카메라 없을 때) |
|---|---|---|
| 붕어 모드 | 입을 'O'로 ~0.4초 유지 | `붕어 모드 ON` 버튼 · `Space`/`O` |
| 캐스팅 | 고개를 빠르고 크게 돌렸다 정면 복귀 (멀수록 비거리↑) | 화면 휙 스와이프 · `살짝 던지기` 버튼 · `Space` |
| 챔질 | 입을 오므렸다('뻐끔') 벌렸다 반복 | `뻐끔!` 버튼 연타 · `Space` 연타 |

- **카메라 거부/모델 로드 실패 시** 자동으로 버튼·키보드 모드로 전환됩니다 → 카메라 없이도 테스트 가능.
- **비거리 ↔ 챔질 난이도 연동**: 멀리(깊게) 던질수록 입질이 빠르고, 필요한 뻐끔 횟수가 늘고,
  제한 시간이 짧아집니다. 대신 레어 확률이 올라갑니다(리스크·리워드).

## 튜닝 포인트

- `js/face.js` → `SENS`(감도별 임계값), 랜드마크 인덱스, 스무딩 계수.
- `js/game.js` → `AMP_MAX`/`SPEED_MAX`(캐스팅 파워 매핑), `hookDifficulty`(난이도 곡선),
  `ITEMS`(아이템·등급·가중치), `TIERS`(등급 점수).
- `js/main.js` → 캐스팅 감지 임계값(`0.045`/`0.02`), 입 'O' 유지 시간(`0.4`), NPC 주기.
- 게임 내 **설정**에서 인식 감도·챔질 난이도(느슨/보통/빡빡)·효과음·배경음을 즉시 조절할 수 있습니다.

## 테스트 체크리스트

- [ ] 로컬 서버로 띄워 카메라 권한 허용 → 캘리브레이션(정면→입 'O') 통과
- [ ] 붕어 모드 → 캐스팅(얕게/깊게) → 입질 → 뻐끔 챔질 → 획득까지 한 사이클
- [ ] 깊게 던졌을 때 뻐끔 횟수↑·시간↓ 확인, 얕게는 여유 확인
- [ ] 어항: 획득 아이템 반영, 등급 필터, 최고 전리품 카드
- [ ] 랭킹: TOP3 시상대 + 내 순위, 획득 시 순위/미니랭킹 갱신
- [ ] 설정: 감도/난이도 변경 반영, 어항 비우기, 캘리브레이션 다시
- [ ] **카메라 차단 상태**에서 버튼/키보드로 전체 루프 동작(폴백)
- [ ] 새로고침 후 어항·전리품이 유지되는지(localStorage)
- [ ] 모바일 폭(≤560px) 레이아웃 확인

## 개인정보

웹캠 영상은 **브라우저(기기) 안에서만** 얼굴 지표 계산에 쓰이고, 저장·전송되지 않습니다.
랭킹/어항도 현재 구현은 전부 로컬(localStorage)입니다.
